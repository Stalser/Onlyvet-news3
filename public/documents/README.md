# Документы OnlyVet

В эту папку нужно добавить файлы документов в различных форматах.

## Поддерживаемые форматы:
- **PDF** (.pdf) — рекомендуется для основных документов
- **Word** (.doc, .docx) — для редактируемых версий
- **Текст** (.txt) — для простых документов

## Как добавить документы:

1. Поместите файлы документов в эту папку (`/public/documents/`)
2. Назовите файлы соответственно:
   - `terms.pdf` — Пользовательское соглашение
   - `privacy.pdf` — Политика конфиденциальности
   - `refund.pdf` — Правила оплаты и возврата
   - `consent.pdf` — Согласие на дистанционную консультацию
   - `disclaimer.pdf` — Дисклеймер
   - `cookies.pdf` — Политика cookies
   - `site-rules.pdf` — Правила сайта

3. Обновите информацию о размерах файлов в `/src/app/documents/page.tsx`:
   ```typescript
   {
     title: 'Пользовательское соглашение',
     download: '/documents/terms.pdf',
     format: 'pdf',
     size: '245 КБ', // укажите актуальный размер
   }
   ```

## Примечание:
- Файлы могут быть в любом формате (pdf, doc, docx, txt)
- На странице документов автоматически отобразится иконка формата
- Размер файла указывается вручную для информирования пользователя
