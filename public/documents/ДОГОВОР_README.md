# Договор на оказание ветеринарных услуг

В эту папку нужно добавить файл договора в формате PDF.

## Как добавить:
1. Создайте файл `terms.pdf` с текстом договора
2. Поместите его в эту папку (`/public/documents/`)
3. Файл будет доступен для скачивания по ссылке `/documents/terms.pdf`

## Что должно быть в договоре:
- Реквизиты исполнителя (толькоvet)
- Права и обязанности сторон
- Порядок оказания услуг
- Стоимость и порядок оплаты
- Ответственность сторон
- Порядок разрешения споров
- Срок действия договора

## Примечание:
На странице записи есть ссылка на договор. Клиент может:
- Открыть договор в новой вкладке
- Скачать для ознакомления
- Распечатать при необходимости
