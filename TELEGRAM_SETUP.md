# 📝 Инструкция по настройке Telegram для уведомлений

## 🎯 Что нужно сделать

Чтобы заявки с формы записи отправлялись в Telegram, нужно создать бота и настроить его.

---

## 📋 Шаг 1: Создать бота в @BotFather

1. Откройте Telegram и найдите **@BotFather**
2. Нажмите **Start** или отправьте `/start`
3. Отправьте команду `/newbot`
4. BotFather попросит придумать имя бота (например, `OnlyVet Bot`)
5. Затем попросит придумать username (должен заканчиваться на `bot`, например: `onlyvet_notify_bot`)
6. BotFather выдаст **токен** вида:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```
7. **Скопируйте токен** — он понадобится для `.env.local`

---

## 📋 Шаг 2: Добавить бота в канал/чат @onlyvet_clinic

1. Откройте ваш канал/чат **@onlyvet_clinic**
2. Зайдите в настройки канала
3. Нажмите **Добавить участника** → **Добавить бота**
4. Найдите вашего бота (по username)
5. Дайте боту права **администратора** (хотя бы на отправку сообщений)

---

## 📋 Шаг 3: Узнать chat_id

### Вариант А: Через @userinfobot
1. Откройте **@userinfobot**
2. Нажмите **Start**
3. Бот покажет ваш **Id** (например: `123456789`)

### Вариант Б: Через @getmyid_bot
1. Откройте **@getmyid_bot**
2. Нажмите **Start**
3. Бот покажет **Your Chat ID**

### Вариант В: Для канала
Если уведомления должны приходить в **канал**, chat_id будет начинаться с `-100`:
```
-1001234567890
```

Чтобы узнать chat_id канала:
1. Отправьте любое сообщение в канал
2. Перешлите это сообщение боту **@getmyid_bot**
3. Бот покажет **Forwarded from Chat ID**

---

## 📋 Шаг 4: Заполнить .env.local

1. Откройте файл `.env.local` в проекте
2. Замените значения:

```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=-1001234567890
```

3. Сохраните файл

---

## 📋 Шаг 5: Проверить работу

1. Запустите проект локально:
   ```bash
   cd ~/onlyvet
   npm run dev
   ```

2. Откройте http://localhost:3000/booking
3. Заполните форму и отправьте
4. Проверьте Telegram — должно прийти уведомление

## ⚠️ Возможные проблемы

### ❌ "Telegram credentials not configured"
**Решение:** Проверьте, что `.env.local` существует и заполнен

### ❌ "400 Bad Request"
**Решение:** Проверьте chat_id — возможно, он неверный или бот не добавлен в канал

### ❌ "401 Unauthorized"
**Решение:** Проверьте токен бота — возможно, он неверный или отозван

### ❌ Заявки не приходят
**Решение:**
1. Проверьте консоль сервера на ошибки
2. Убедитесь, что бот добавлен в канал как администратор
3. Отправьте любое сообщение в канал — иногда это требуется для активации
