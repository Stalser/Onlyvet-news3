## 🚀 КАК ПОДКЛЮЧИТЬ БД

### 1. Выбрать вариант:

| Вариант | Сложность | Vercel | Рекомендация |
|---------|-----------|--------|--------------|
| **PocketBase** | 🟢 Низкая | ✅ Да | ⭐ Рекомендуется |
| **PostgreSQL** | 🟡 Средняя | ✅ Да | Для роста |
| **SQLite** | 🟢 Низкая | ❌ Нет | Только локально |

### 2. Инструкция:

См. **`DATABASE_SETUP.md`** — полная инструкция по каждому варианту.

### 3. Переключение:

После настройки БД:
```typescript
// src/lib/api.ts
const USE_API = true; // было false
```
